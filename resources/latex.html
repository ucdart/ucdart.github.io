Refer to the Wikipedia entry for basic information on LaTeX. Some great introductory tutorials on LaTeX include The Not So Short Introduction to LaTeX2e by Tobi Oetiker and the LATEX Tutorials -- A PRIMER by the Indian Tex User Group. The LaTeX Wikibook is also a great start point. Intermediate level users may find "Guide to LaTeX: Tools and Techniques for Computer Typesetting" by Helmut Kopka and Patrick W. Daly useful.
